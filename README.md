The project is a simple to-do application made with Django, created with a goal of practicing HTML, CSS as well as connection of the application to the database (Moved from MySQL to SQLite in order to have a functional project without the need to set up the database).

The functionality includes:
  The ability to register an account to store your tasks - each accounts tasks are only visible to that account;
  Visibility of the number of incomplete tasks;
  The ability to mark tasks as completed or in progress, the completed tasks get crossed out and are moved to the bottom of the screen;
  The search box;
